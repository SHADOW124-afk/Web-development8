What is HTML Code?

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It provides the structure of a website, defining elements like headings, paragraphs, images, and links.

Uses of HTML Code

  • Creating the basic structure of a website
  • Adding text, images, and videos to a webpage
  • Linking different pages together
  • Embedding multimedia content

HTML code is written using tags, which are enclosed in angle brackets (< >). These tags tell the browser how to display the content of the webpage.

HTML

The <html> tag is the root element of an HTML page.

How it will look

The <html> tag itself doesn't change how content is displayed, but it's essential for the browser to understand that it's reading an HTML document.

Head

The <head> element contains meta-information about the HTML document, such as the title of the page.

How it will look

The <head> content isn't visible on the page itself, but the title will appear in the browser's title bar or tab.

Body

The <body> element contains the visible page content.

How it will look

Everything inside the <body> tag will be displayed on the webpage.

Title

The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).

How it will look

The <title> content appears in the browser's title bar or tab.
