Steps in Web Development (Mobile)

  1. File Manager: You will need a file manager. You can download ZArchiver or MT Manager file manager and use it for making a website. You can also use your built-in file manager in your Android or iOS device, but I recommend you to use ZArchiver and MT Manager.
  2. Write HTML Code: If you have used ZArchiver and MT Manager, open MT Manager and create a folder with your first website.html and start typing the codes in your HTML file.
  3. Save HTML Page: After you have written the codes in the MT Manager, you will see a save button. Click on the save button and your file will be saved.
  4. View HTML Page in Browser: Now, if you want to view it in the browser, open ZArchiver and go to the directory where you have saved the HTML file. Click on the file, a menu will appear. Click on the open button, a new menu will appear asking you where to open it. Click on chrome and your HTML page will be opened.
