Steps in Web Development

Creating a simple web page using HTML is very easy. It requires a text editor, a file with HTML extension, and a web browser to view that page.

  1. STEP 1: TEXT EDITOR Start by simply creating new blank files in a text editor of your choice. A simple text editor like Notepad can be used to start coding HTML for a web page.
  2. STEP 2: WRITE HTML CODE IN TEXT EDITOR Example HTML: 
    <html>
<head>
    <title>My website</title>
</head>
<body>
Welcome to my first web page
</body>
</html>
  3. STEP 3: SAVE HTML PAGE Go to the file menu and click on save. Make sure to provide .html or .htm extension for the file being saved. This will save the document as a web page instead of a plain text file.
  4. STEP 4: VIEW HTML PAGE IN BROWSER Open the saved HTML file in your default web browser. The web browser will automatically translate HTML codes to correctly display the webpage.
